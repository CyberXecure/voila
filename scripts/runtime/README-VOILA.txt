VOILA! - README

Voila! este o aplicație locală pentru transformarea manualelor PDF în conținut de studiu.

Această versiune este un pachet standalone ZIP. Nu necesită instalare clasică.

Include:
- Python runtime local
- Tesseract OCR local
- Java runtime local
- LanguageTool local
- scripturi de pornire/oprire
- scripturi pentru creare/ștergere shortcut

Pornire rapidă:
1. Extrage ZIP-ul.
2. Intră în folderul app.
3. Rulează Create-Voila-Shortcut.cmd.
4. Pornește Voila din shortcut-ul de pe Desktop.

Adresa locală:
http://127.0.0.1:8765

Health check:
http://127.0.0.1:8765/health

Fișiere utile:
- START-AICI.txt
- QUICKSTART.txt
- TROUBLESHOOTING.txt
- Run-Voila.cmd
- Stop-Voila.ps1
- Create-Voila-Shortcut.cmd
- Remove-Voila-Shortcut.cmd

Notă:
Voila rulează local pe calculatorul tău. Pachetul nu modifică PATH-ul global și nu instalează servicii Windows.

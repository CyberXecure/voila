# Voila! v0.2.0 Public Beta

Voila! v0.2.0-public-beta is the first public beta release prepared for GitHub distribution.

This release is based on the validated v0.1.9-final-release-polish standalone package and focuses on providing a clean, user-facing public beta experience.

## What is included

- Standalone Windows ZIP package
- Embedded Python runtime
- Embedded Tesseract OCR runtime
- Embedded Java runtime
- Embedded LanguageTool runtime
- Professional application icon
- Desktop shortcut support
- Start Menu shortcut support
- Create-Voila-Shortcut helper
- Remove-Voila-Shortcut helper
- User-facing documentation included in the ZIP:
  - START-AICI.txt
  - QUICKSTART.txt
  - TROUBLESHOOTING.txt
  - README-VOILA.txt

## Validation status

The package was validated through:

- Automated standalone runtime smoke test
- Manual user-flow test
- Runtime dependency verification
- Shortcut creation/removal verification
- Documentation review
- SHA256 checksum generation

## Recommended usage

Download the ZIP package, extract it to a local folder, then open the included START-AICI.txt file.

This is a public beta release. Feedback, bug reports, and improvement suggestions are welcome.

## Stability note

The runtime package used for this public beta is derived from the internally validated v0.1.9-final-release-polish release candidate.

## Summary

This milestone improves Study concept/title cleanup and prevents stale concept cards from remaining visible after Study quiz regeneration.

## Implemented changes

- Added concept-title cleanup helpers in `study_quiz_builder.py`.
- Cleans noisy generated concept titles before writing `quiz.study.json`.
- Filters common attribution/source fragments from generated concept titles, including:
  - Department of Energy
  - Wikimedia Commons
  - NASA/Ames
  - modification of work
  - credit/source/courtesy fragments
  - figure/table/caption-like fragments
  - trailing punctuation artifacts
- Added defensive concept cleanup in `study_questions.py` for display of existing/generated questions.
- Updated Study state loading in `study_engine.py` so `study_state.json` is synchronized with the current `quiz.study.json`.
- Removes stale concepts, stale attempts, and stale last-attempt data when the current quiz no longer contains those question/concept IDs.
- Preserves valid mastery state for concepts that still exist in the current quiz.
- Keeps manual concept override flow available for fallback titles that need user refinement.

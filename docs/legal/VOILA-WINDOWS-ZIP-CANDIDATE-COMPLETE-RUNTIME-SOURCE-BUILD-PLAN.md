# Voila Windows ZIP Candidate Complete Runtime Source Build Plan

Milestone:

```text
v0.3.54-voila-windows-zip-candidate-complete-runtime-source-build-plan
```

## Purpose

Plan the first Windows ZIP candidate build that uses a complete runtime source generated by:

```text
scripts/release/create-complete-windows-runtime-source.ps1
```

This milestone is documentation/release build planning only.

It does not:

```text
create a runtime source
copy runtime files
rebuild a package
create a ZIP
create SHA256
extract a ZIP
create an installer
run START-VOILA.bat
run STOP-VOILA.bat
publish a GitHub release
upload release assets
change GitHub visibility
add payment or activation
provide final legal approval
```

## Planned build flow

Recommended local flow:

```text
1. sync protected main
2. verify clean working tree
3. create temporary complete runtime source
4. run create-complete-windows-runtime-source.ps1
5. validate runtime source summary
6. run build-windows-zip-candidate.ps1
7. copy legal files during build
8. validate package staging with -Strict
9. create ZIP
10. generate SHA256
11. extract ZIP for validation
12. verify real complete runtime content in extracted ZIP
13. verify no EXE/MSI installer created
14. document build result
```

---

## Planned commands

Create complete runtime source:

```powershell
.\scripts\release\create-complete-windows-runtime-source.ps1 `
  -RuntimeSourceRoot <runtime-source-root> `
  -PythonStrategy PackageVenv `
  -IncludeCropEditor `
  -LanguageToolStrategy Deferred `
  -OcrStrategy Deferred `
  -Force
```

Build ZIP candidate:

```powershell
.\scripts\release\build-windows-zip-candidate.ps1 `
  -RuntimeSource <runtime-source-root>\voila `
  -OutputRoot <output-root> `
  -Version "v0.3.54" `
  -ReleaseType PublicBeta `
  -Force
```

# Third-Party Notices

Voila may include or depend on third-party software, runtimes, tools, libraries, language resources, OCR tools, and other components.

This file is a notice and inventory placeholder for third-party components used by Voila public beta and tester packages.

Before commercial distribution, this file should be reviewed and completed with exact component names, versions, copyright notices, license names, and license texts or links.

## Known component categories to review

```text
Python runtime and Python packages
FastAPI and backend dependencies
Frontend dependencies
OCR tooling
Tesseract-related files
LanguageTool
Java / JRE runtime
PDF processing libraries
Packaging scripts and bundled runtime tools
Fonts, icons, screenshots, and public assets
```

## Policy

Third-party components remain under their respective licenses.

Voila does not claim ownership of third-party components.

The Voila proprietary license notice applies only to original Voila project code, documentation, branding, screenshots, release packaging, and project-specific materials, unless stated otherwise.

## Commercial-readiness checklist

Before any paid release or commercial package:

```text
[ ] list all bundled third-party components
[ ] list exact versions
[ ] identify each component license
[ ] include required notices
[ ] include required license texts or links
[ ] verify redistribution rights
[ ] verify attribution requirements
[ ] verify source-code-offer obligations, if any
[ ] verify bundled runtime obligations
[ ] verify icon/font/media rights
[ ] review with legal counsel if needed
```

## Purpose

Define the short tester instruction asset to include with the staged package.

## Planned README-TESTERS-SHORT.txt

Recommended content:

```text
Voila Windows Public Beta - Tester Instructions

1. Extract the ZIP to a normal folder, for example Desktop or Downloads.
2. Open the extracted voila folder.
3. Run START-VOILA.bat.
4. Open http://127.0.0.1:8787 in your browser.
5. Test with a small non-private PDF.
6. When finished, run STOP-VOILA.bat.
7. Send feedback using FEEDBACK-TEMPLATE.txt.

Do not test with confidential, medical, financial, legal, customer, or identity documents.

This is a beta test package. It is not final production software.
```

- Added BKT diagnostics section inside Progress concept cards.
- Displays per-concept BKT metadata:
  - recent misses
  - last question type
  - last correct timestamp
  - last incorrect timestamp
  - question-type stats
- Added recommended next focus details:

  - next question type
- Localized Progress BKT diagnostics labels for Romanian and English.
- Refined Romanian mastery wording:
  - Mastered -> Consolidat
  - Almost mastered -> Aproape consolidat
  - Overall mastery -> Nivel general
  - Concept mastery -> Nivelul conceptelor
  - mastery -> nivel
- Localized Progress coverage labels:
  - Answered -> Răspunse
  - Total questions -> Total întrebări
- Cleaned duplicate Romanian runtime translation for recommended_because.

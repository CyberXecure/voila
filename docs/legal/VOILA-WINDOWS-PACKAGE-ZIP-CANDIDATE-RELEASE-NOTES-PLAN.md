# Voila Windows Package ZIP Candidate Release Notes Plan

## Purpose

Define release notes content for the future Voila Windows ZIP candidate.

This is planning-only.

---

## Release notes filename

Inside package:

```text
RELEASE-NOTES.txt
```

Optional external asset:

```text
voila-v0.3.25-public-beta-windows-package-candidate-release-notes.txt
```

---

## Required fields

```text
Release type:
Version:
Date:
Intended audience:
Runtime package:
Candidate status:
Known limitations:
Page-count limits:
Commercial use:
Redistribution:
Legal files:
SHA256:
Smoke test status:
Support/contact:
```

---

## Candidate wording

Recommended wording:

```text
This is a Windows package candidate for validation. It is not a final commercial release.
```

---

## Legal wording

Recommended wording:

```text
Before using this package, review the legal/ folder.

Included legal files:
- legal/EULA.txt
- legal/LICENSE.txt
- legal/BETA-TERMS.md
- legal/THIRD-PARTY-NOTICES.md
```

---

## Commercial use wording

For current beta/candidate status:

```text
Commercial use is not enabled by this candidate unless explicitly allowed in written terms.
```

---

## Redistribution wording

Recommended:

```text
Do not redistribute, repackage, sell, or publish this candidate unless explicit written permission is provided.
```

---

## SHA256 wording

Before checksum generation:

```text
SHA256: to be generated after ZIP candidate creation.
```

After checksum generation:

```text
SHA256: <hash>
```

---

## Smoke status wording

Before smoke:

```text
Smoke test status: pending.
```

After smoke:

```text
Smoke test status: PASS / CONDITIONAL / FAIL.
```

---

## Limitations section

Include:

```text
public beta limitations
known OCR limitations
supported document types
page-count limits if any
local-only/runtime notes
no final legal/commercial guarantee
```

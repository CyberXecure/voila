## 1. Introduction

This Public Beta End User License Agreement draft applies to Voila public beta, tester demo, and related Windows package builds unless a separate written agreement says otherwise.

Voila is proprietary beta software.

By downloading, installing, extracting, running, or using a Voila beta package, the user agrees to the applicable beta terms, license notice, release notes, and this draft agreement once it is formally included with a package.

This draft should be reviewed before being used as final package terms.

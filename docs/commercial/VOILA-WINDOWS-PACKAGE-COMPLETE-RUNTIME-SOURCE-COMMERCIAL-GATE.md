# Voila Windows Package Complete Runtime Source Commercial Gate

Milestone:

```text
v0.3.47-voila-windows-package-complete-runtime-source-plan
```

## Purpose

Define commercial/release boundaries for moving to a complete runtime source.

---

## Gate before complete runtime build

Required:

```text
[ ] API entrypoint identified
[ ] dependency strategy selected
[ ] frontend/static strategy selected
[ ] LanguageTool/Java decision made
[ ] Tesseract/OCR decision made
[ ] runtime source exclusions defined
[ ] complete runtime validation checklist ready
```

---

## Gate before external tester sharing

Required later:

```text
[ ] complete runtime ZIP build PASS
[ ] START/STOP smoke PASS or clearly documented CONDITIONAL
[ ] known limitations documented
[ ] README reviewed
[ ] release notes reviewed
[ ] legal folder verified
[ ] feedback/support path defined
```

---

## Gate before commercial distribution

Required later:

```text
[ ] paid terms ready
[ ] EULA reviewed for paid use
[ ] third-party notices complete enough for distribution
[ ] license/activation decision made
[ ] refund/support wording reviewed
[ ] external dependency obligations reviewed
```

---

## Boundary

A complete runtime source does not automatically mean:

```text
final release
commercial readiness
legal approval
public release approval
paid distribution approval
```

# Voila Windows ZIP Candidate Real Launchers Commercial Gate

## Purpose

Define commercial/release gate for replacing placeholder launchers with real launchers.

This is guidance only.

---

## Gate before implementation

Required:

```text
[ ] real launchers plan complete
[ ] real launchers spec complete
[ ] validation plan complete
[ ] runtime package layout confirmed
[ ] process ownership strategy selected
```

---

## Gate before runnable ZIP claim

Required:

```text
[ ] real launchers implemented
[ ] ZIP candidate rebuilt
[ ] SHA256 regenerated
[ ] START/STOP smoke PASS
[ ] local service responds
[ ] STOP cleans up package-owned processes
[ ] README/release notes updated
```

---

## Gate before sharing externally

Required:

```text
[ ] runnable ZIP smoke PASS
[ ] known limitations clear
[ ] legal folder verified
[ ] feedback/support path defined
[ ] candidate status clear
```

---

## Commercial boundary

Real launchers improve package readiness but do not alone make Voila commercially ready.

Before paid Supporter/Pro distribution:

```text
[ ] paid terms ready
[ ] EULA reviewed for paid use
[ ] third-party notices complete enough for distribution
[ ] payment/refund wording reviewed
[ ] license/activation plan decided
```

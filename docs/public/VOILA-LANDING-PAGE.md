# Voila! Landing Page Draft

## Headline

Turn PDFs into structured learning materials.

## Subheadline

Voila! is a public beta tool for converting PDF documents into course outlines, cleaned course content, glossary entries, quizzes, flashcards, and OCR review outputs.

## Current status

Voila! v0.3.0 Public Beta — Language Pack RC1 is available as a GitHub prerelease.

The RC1 package focuses on language-pack infrastructure, English/Romanian coverage, validation, release-readiness documentation, and a safer path toward future localization.

## Who it is for

Voila! may be useful for:

- students
- teachers
- trainers
- self-learners
- people converting PDF materials into study content
- people reviewing OCR output before generating learning materials

## What it does

Voila! can generate:

- course outline
- normalized outline
- course markdown
- cleaned course markdown
- glossary
- quiz
- flashcards
- OCR review artifacts
- language-pack validation assets

## Language-pack status

Current RC1:

- English
- Romanian

Future candidates:

- German

- Portuguese

Future languages will be prioritized based on feedback, not forced early.

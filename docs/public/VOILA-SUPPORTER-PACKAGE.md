# Voila! Supporter Package Draft

## Purpose

The supporter package is a lightweight monetization option before deciding on a full commercial model.

It should support development without forcing a complex license or paid SaaS model too early.

## Possible tiers

### Community / Public Beta

Included:

- public GitHub prereleases
- basic documentation
- English and Romanian language-pack RC1
- public feedback channel

Not included:

- priority support
- custom language packs
- guaranteed fixes
- commercial support

### Supporter Package

Possible benefits:

- easier packaged build
- quick-start PDF-to-course guide
- example workflows
- supporter-only notes or templates
- priority bug triage
- early access to future language-pack candidates
- optional public thank-you mention

### Future Pro / Commercial

Possible future benefits:

- polished installer
- additional language packs
- better export templates
- batch workflows
- support for trainers or educators
- commercial usage terms
- priority support

## Current recommendation

Do not launch paid tiers immediately.

First validate:

- whether users understand Voila!
- whether they test it
- whether they ask for support
- whether they care about language packs
- whether they would pay for convenience, support, or extra templates

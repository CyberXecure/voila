# Voila! Language Expansion Plan

## Current RC1 languages

Included:

- English
- Romanian

## Future language candidates

Planned / future expansion:

- German
- Spanish
- Italian
- Portuguese

## Recommendation

Do not force German, Spanish, Italian, or Portuguese into RC1.

Reason:

- English and Romanian are enough to validate the language-pack mechanism.
- More languages increase maintenance cost.
- Translation quality should be validated by real users.
- User demand should decide priority.

## Expansion criteria

Add a new language only when at least one of these is true:

- users ask for it
- a reviewer can validate translation quality
- a real testing workflow exists for that language
- the language helps monetization or adoption
- the language pack can be validated without breaking runtime behavior

## Proposed order

Initial recommendation:

1. English
2. Romanian
3. Spanish
4. German
5. Italian
6. Portuguese

This order can change based on feedback.

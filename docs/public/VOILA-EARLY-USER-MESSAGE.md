# Voila! Early User Message

I published a release candidate for Voila!, a tool that turns PDF documents into learning materials such as course outlines, cleaned course content, glossary entries, quizzes, flashcards, and OCR review outputs.

The latest prerelease includes Language Pack RC1 with initial English/Romanian support, a language-pack schema, validator, sample packs, runtime helper, release-readiness documentation, and verified release assets.

I am looking for early feedback from a small number of testers.

The most useful feedback right now:

- Is it clear what Voila! does?
- Who would this be useful for?
- Is PDF-to-course generation valuable?
- Which output matters most: course, glossary, quiz, flashcards, OCR review?
- Would English/Romanian be enough for the first public beta?
- Which language should be next: German, Spanish, Italian, or Portuguese?
- Would a paid supporter package make sense?

I am not looking for big feature requests yet. The goal is to validate whether the product direction is useful before building more.

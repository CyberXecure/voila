# v0.6.3 Controlled Tester Package Contents Manifest — No Build, No Distribution

Status: implementation milestone
Scope: owner-controlled contents manifest only
Build: explicitly blocked
Distribution: explicitly blocked

## Purpose

This milestone records the intended contents of a future controlled tester package without building, zipping, staging a release archive, or distributing anything.

It does not create a runnable package.

## Candidate future package contents

The future package may be planned around these categories only:

- application source/runtime files required by the approved tester package process
- local launcher scripts approved by a later milestone
- tester-facing README and limitations documents approved by a later milestone
- proprietary license/terms files approved by a later milestone
- local-only validation report generated by a later milestone

This milestone does not approve the inclusion, copying, building, packaging, signing, uploading, or sharing of any of those contents.

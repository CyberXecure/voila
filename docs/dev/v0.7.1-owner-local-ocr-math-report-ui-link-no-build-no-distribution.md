# v0.7.1-owner-local-ocr-math-report-ui-link-no-build-no-distribution

## Scope

This owner-local milestone adds a visible UI link/card for the existing OCR Math diagnostic report.

When `ocr_math_report.md` exists for a local course/document, Voila can show a small local card/link to:

- `ocr_math_report.md`
- `total_suggestions`
- `changed_line_count`

## Policy

This milestone is UI/link-only.

It does not:

- auto-correct formulas
- rewrite OCR text
- rewrite pages
- rewrite course content
- change Study
- change Progress
- add Formula OCR
- build packages
- create ZIP files
- deliver files
- distribute files

The existing diagnostic generation policy remains unchanged: `Generate` / `Regenerează` report creation remains gated by `VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1`.

## Implementation

Added owner-local routes:

- `/owner/ocr-math-report/{course_id}/summary.json`
- `/owner/ocr-math-report/{course_id}/ocr_math_report.md`

Added a small owner-local HTML injection that displays a report card/link on course/document related pages and library/home cards when a matching report exists.

## Validation

Run:

```powershell
pwsh -NoProfile -ExecutionPolicy Bypass -File scripts/dev/check-v0.7.1-owner-local-ocr-math-report-ui-link-no-build-no-distribution.ps1
```

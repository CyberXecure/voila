# v0.7.3-owner-local-ocr-math-report-viewer-page-no-build-no-distribution

## Scope

This owner-local milestone adds a dedicated read-only viewer page for the existing OCR Math diagnostic report.

New local viewer route:

- `/owner/ocr-math-report/{course_id}/view`

The viewer page:

- displays a readable owner-local HTML view of `ocr_math_report.md`
- keeps a direct link to the raw Markdown report
- shows summary metrics at the top:
  - `Sugestii detectate`
  - `Linii posibil afectate`
  - `Diagnostic local · read-only`
- safely escapes Markdown content before rendering
- uses a small dependency-free Markdown subset renderer
- shows a clear 404 page when no report exists

The existing v0.7.1 routes remain available:

- `/owner/ocr-math-report/{course_id}/summary.json`
- `/owner/ocr-math-report/{course_id}/ocr_math_report.md`

The v0.7.2 card now links to the viewer page instead of opening the raw Markdown directly.

## Policy

This milestone is read-only viewer UI only.

It does not:

- edit the report
- accept suggestions
- reject suggestions
- auto-correct formulas
- add Formula OCR
- rewrite OCR text
- rewrite pages
- rewrite course content
- change Study
- change Progress
- build packages
- create ZIP files
- deliver files
- distribute files

The existing diagnostic generation policy remains unchanged: `Generate` / `Regenerează` report creation remains gated by `VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1`.

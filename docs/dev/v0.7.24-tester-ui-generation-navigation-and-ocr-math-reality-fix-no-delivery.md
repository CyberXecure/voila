## Root cause

Extracted-package testing of the v0.7.20 tester package showed that `/generate` produced the real Markdown course artifacts, including `course.md` and `course.cleaned.md`, but did not reliably produce `course.cleaned.html`.

The homepage and some tester UI entry points treated `course.cleaned.html` as the only source of truth for a generated course. Therefore a course with valid generated Markdown could still be displayed as `Negenerat încă`, and the tester could not see the expected `Deschide cursul` path.

Course Tools also used stale availability assumptions. Some cards appeared disabled even when useful artifacts or routes existed. OCR Math was present only through owner-local hidden routes and was not visible as a clear tester-facing diagnostic option.

## Fixed behavior

- `generate_for_pdf()` now calls `ensure_course_html_for_pdf()` after the Markdown generation pipeline.
- `ensure_course_html_for_pdf()` exports `course.cleaned.html` from `course.cleaned.md` using the existing `html_exporter.py` path and injects course navigation through `course_nav_injector.py`.
- Homepage detection now treats `course.cleaned.md` as generated content even when `course.cleaned.html` is missing.
- Homepage shows a truthful `Generated · HTML pending` state instead of falsely showing `Negenerat încă`.
- `/course-open?pdf=...` rebuilds `course.cleaned.html` on demand when only `course.cleaned.md` exists, then redirects to the generated HTML course.
- Course Tools now exposes real tester-flow actions for: Bibliotecă, Instrumente curs, Deschide cursul, Studiu, Progres, OCR Review, Exam Prep, OCR Math Diagnostic, figures/crops and local JSON artifacts when available.
- OCR Math Diagnostic is visible in Course Tools. If `ocr_math_report.md/json` is missing, the card explains that the report has not been generated and is only created when the local OCR Math hook is active.
- Important tester-flow pages now have bottom navigation through either the shared `appFixedNav`, explicit Course Tools navigation, or the standalone HTML middleware injection.

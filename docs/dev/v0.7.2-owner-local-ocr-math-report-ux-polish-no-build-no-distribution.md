# v0.7.2-owner-local-ocr-math-report-ux-polish-no-build-no-distribution

## Scope

This owner-local milestone polishes the visible OCR Math diagnostic report UI added in v0.7.1.

The milestone improves only the UI card/link for an existing local diagnostic report:

- clearer Romanian title: `Raport diagnostic OCR Math disponibil`
- local-only badge: `Diagnostic local`
- friendlier metric labels:
  - `Sugestii detectate`
  - `Linii posibil afectate`
- clearer helper text explaining that the report is informational only
- more polished card spacing, border, background, and link styling
- no card is shown when the report summary endpoint returns 404 or `exists=false`

## Policy

This milestone is UX polish only.

It does not:

- auto-correct formulas
- add Formula OCR
- rewrite OCR text
- rewrite pages
- rewrite course content
- change Study
- change Progress
- build packages
- create ZIP files
- deliver files
- distribute files

The v0.7.1 owner-local routes remain the same:

- `/owner/ocr-math-report/{course_id}/summary.json`
- `/owner/ocr-math-report/{course_id}/ocr_math_report.md`

The existing diagnostic generation policy remains unchanged: `Generate` / `Regenerează` report creation remains gated by `VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1`.

## Validation

Run:

```powershell
pwsh -NoProfile -ExecutionPolicy Bypass -File scripts/dev/check-v0.7.2-owner-local-ocr-math-report-ux-polish-no-build-no-distribution.ps1
```

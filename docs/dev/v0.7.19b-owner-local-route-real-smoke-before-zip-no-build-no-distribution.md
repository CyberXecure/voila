## Hotfix note

During v0.7.19b local route smoke, the OCR Math summary route exposed a slow path in `_voila_ocr_math_report_paths`: controlled reports under `data/output/{course_id}` were found only after recursive scanning.

This milestone adds direct fast-path candidates for:

```text
root / "output" / course_id
root / "data" / "output" / course_id
```

This keeps OCR Math summary/markdown/viewer route resolution local and bounded before tester ZIP preparation.

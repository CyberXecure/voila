# v0.7.0 OCR Math Report Hook

Scope: owner-local only. No build, no ZIP, no delivery, no public upload, no distribution.

This milestone adds a safe hook wrapper around the OCR math diagnostic report.

## What it does

The hook can generate:

- `ocr_math_report.json`
- `ocr_math_report.md`

inside a PDF output folder when explicitly enabled.

## Safety policy

The hook does not modify:

- OCR source text
- `pages.md`
- `course.md`
- `course.cleaned.md`
- quiz files
- Study
- Progress
- public UI
- release/package/delivery assets

## Enable policy

The hook is disabled unless explicitly enabled by code, CLI flag, or owner-local environment flag:

```text
VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1
```

## Why

This gives Generate a safe future integration point without changing behavior yet.

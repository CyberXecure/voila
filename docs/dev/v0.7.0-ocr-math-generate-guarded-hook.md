# v0.7.0 Guarded OCR Math Generate Hook

Scope: owner-local only. No build, no ZIP, no delivery, no public upload, no distribution.

This milestone wires the OCR math report hook into `generate_for_pdf`, but only as a guarded diagnostic hook.

## Enable policy

The hook remains disabled unless explicitly enabled:

```text
VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1
```

## What it can create

When enabled, Generate/Regenerează may create:

- `ocr_math_report.json`
- `ocr_math_report.md`

inside the PDF output folder.

## What it does not do

It does not modify:

- OCR text
- `pages.md`
- `pages.json`
- `course.md`
- `course.cleaned.md`
- quiz files
- Study
- Progress
- build/package/release/delivery assets

## Failure policy

The hook is wrapped in a defensive `try/except`; it must never break Generate/Regenerează.

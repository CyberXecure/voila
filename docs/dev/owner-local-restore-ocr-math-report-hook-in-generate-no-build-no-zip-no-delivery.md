## Purpose

This milestone restores the owner-local OCR Math diagnostic report hook inside the real `generate_for_pdf()` pipeline.

Before this fix, the OCR Math UI card correctly stayed unavailable because `ocr_math_report.md` and `ocr_math_report.json` were not generated by `/generate`, even when `VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1` was set.

## Change

`web_app.py` now calls `ocr_math_report_hook.py` after `pdf_extract.py` and before outline/course generation.

The hook is still explicitly gated by:

`VOILA_ENABLE_OCR_MATH_REPORT_HOOK=1`

When enabled, it writes only:

- `ocr_math_report.json`
- `ocr_math_report.md`
